# Ошибки 

### BUG-01: Создание студента с пустым именем
- Приложение позволяет добавить студента без имени.
- Приоритет: High
- Статус: Open

### BUG-02: Принимаются некорректные оценки
- Можно ввести значение вне диапазона (например, -1 или 200).
- Приоритет: High
- Статус: Open

### BUG-03: Нет понятных сообщений об ошибках
- При некорректном вводе сообщения отсутствуют или не объясняют проблему.
- Приоритет: Medium
- Статус: Open
