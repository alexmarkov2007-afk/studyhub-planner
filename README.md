
#  StudyHub Planner

**StudyHub Planner** — учебное веб-приложение на React для планирования учебных задач. Позволяет студентам отслеживать дедлайны, отмечать выполненные задания и структурировать учебный процесс.

---

##  1. Описание проекта

**StudyHub Planner** — это небольшое приложение.
Основная цель проекта — помочь студентам удобным способом организовать свои задания, а также продемонстрировать навыки работы с React, компонентами, состоянием и локальным хранилищем браузера.

**Функциональность:**

* добавление задач (название, предмет, дедлайн, приоритет);
* отображение списка задач;
* фильтрация по статусу: *все / активные / выполненные*;
* фильтрация по предметам;
* отметка задачи как выполненной;
* удаление задач;

**Целевая аудитория:**
студенты, учащиеся и начинающие разработчики, изучающие React.

---

##  2. Установка и запуск

##  Как начать пользоваться приложением

После запуска приложения в браузере перед вами откроется главная страница **StudyHub Planner**. Чтобы приступить к работе, выполните следующие шаги:

### 1. Добавление задачи

1. В верхней части страницы найдите форму **«Добавить задачу»**.
2. Укажите:

   * **Название задачи** — кратко опишите, что нужно сделать.
   * **Предмет** — выберите дисциплину из списка.
   * **Дедлайн** — установите дату выполнения.
   * **Приоритет** — выберите уровень важности.
3. Нажмите кнопку **«Добавить»** — задача появится в общем списке.


### 2. Просмотр списка задач

После добавления задания оно отображается в блоке **«Список задач»**.
Каждый элемент содержит:

* название задания,
* предмет,
* дату дедлайна,
* приоритет,
* кнопки управления (выполнить / удалить).



### 3. Управление задачами

####  Отметить задачу выполненной

Нажмите кнопку **«Выполнено»** — задача будет перечёркнута и перемещена в список выполненных.

####  Вернуть задачу в работу

Если вы отметили задачу ошибочно, нажмите **«Вернуть»**.

####  Удалить задачу

Нажмите кнопку **«Удалить»**, чтобы убрать задачу из списка.



### 4. Фильтрация задач

В верхней панели доступны фильтры:

* **Все задачи** — показываются все записи.
* **Активные** — только задачи, которые ещё нужно выполнить.
* **Выполненные** — только завершённые задачи.
* **По предмету** — можно выбрать конкретную дисциплину.

Фильтры помогают быстро ориентироваться, когда задач становится много.


### 5. Сохранение данных


* задания **не исчезнут после перезагрузки страницы**;
* данные хранятся только на вашем устройстве;
* ничего не требуется дополнительно настраивать.




##  3. Примеры использования

### Добавление задачи

1. Введите название задания
2. Выберите предмет
3. Укажите дату дедлайна
4. Нажмите **«Добавить»**

Задача появится в общем списке.

### Пример React-кода

```jsx
function TaskItem({ task, onToggle, onDelete }) {
  return (
    <li className={`task ${task.completed ? "task--done" : ""}`}>
      <div>
        <strong>{task.title}</strong> — {task.subject}
        <div>Дедлайн: {task.deadline}</div>
      </div>

      <div className="task__actions">
        <button onClick={() => onToggle(task.id)}>
          {task.completed ? "Вернуть" : "Выполнено"}
        </button>
        <button onClick={() => onDelete(task.id)}>Удалить</button>
      </div>
    </li>
  );
}
```
